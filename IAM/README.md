# 🔐 AWS IAM Lab

Identity and Access Management (IAM) is the foundation of security in AWS. This lab demonstrates my hands-on implementation of IAM concepts, focusing on secure access management, permission control, and AWS security best practices.

## 🚀 What I Practiced

* Creating IAM Users, Groups, and Roles
* Working with AWS Managed & Customer Managed Policies
* Implementing Least Privilege Access
* Configuring MFA and Password Policies
* Managing Access Keys
* Testing permissions using the IAM Policy Simulator
