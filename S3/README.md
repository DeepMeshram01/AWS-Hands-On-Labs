# 🪣 AWS S3 Lab

Amazon S3 (Simple Storage Service) is AWS's highly scalable object storage service used for storing and managing data securely. This lab focuses on implementing S3 features, exploring storage management, access control, and data protection through hands-on practice.

## 🚀 What I Practiced

* Creating and Managing S3 Buckets
* Uploading and Organizing Objects
* Bucket Policies and Access Control
* Versioning and Object Lifecycle Management
* Static Website Hosting
* Server-Side Encryption
* Storage Classes
* Pre-Signed URLs
