# 💻 AWS EC2 Lab

Amazon EC2 (Elastic Compute Cloud) provides scalable virtual servers in the cloud for deploying and managing applications. This lab focuses on launching, configuring, and securing EC2 instances while exploring compute, networking, storage, and remote access through hands-on practice.

## 🚀 What I Practiced

* Launching and Configuring EC2 Instances
* Working with Amazon Machine Images (AMIs)
* Creating and Managing Key Pairs
* Configuring Security Groups
* Connecting via SSH and EC2 Instance Connect
* Attaching and Managing EBS Volumes
* Creating AMIs and Snapshots
* Monitoring Instance Performance
